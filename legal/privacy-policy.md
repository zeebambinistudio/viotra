# Kebijakan Privasi (Privacy Policy) - Viotra

*Berlaku sejak: 27 Januari 2026*

Kami menghargai privasi Anda, dokumen ini menjelaskan bagaimana kami mengelola data Anda.

## Data yang Kami Kumpulkan
- Informasi Identitas: Nama dan alamat email.
- Data Keuangan: Catatan transaksi, dompet yang Anda input secara mandiri, kategori keuangan, anggaran, target, liabilitas, dan catatan tugas.
- Data Teknis: Log error dan informasi perangkat untuk keperluan perbaikan bug.

## Cara Kami Menggunakan Data
- Data keuangan Anda HANYA digunakan untuk menampilkan grafik, laporan, dan fitur dalam aplikasi Viotra.
- Kami TIDAK menjual/memberi data pribadi atau data keuangan Anda kepada pihak ketiga manapun untuk tujuan iklan.

## Layanan Pihak Ketiga
- Google Firebase: Digunakan untuk menyimpan data (Cloud FIrestore) dan analitik.
- Data Anda disimpan di server yang dikelola oleh Google dengan standar keamanan enkripsi internasional.

## Keamanan Data
- Komunikasi data dienkripsi menggunakan SSL/TLS (HTTPS).
- Akses database dibatasi hanya untuk pemilik akun yang sah (sesuai verifikasi login).

## Hak Pengguna
- Anda berhak meminta penghapusan akun dan seluruh data terkait kapan saja melalui fitur dalam aplikasi dan/atau melalui narahubung di bawah.
- Anda dapat mengubah data profil sesuai fitur yang disediakan.
